# 项目总览

本项目用于支撑毕业设计《基于 C++ 高并发架构与语言大模型的交互系统的开发》。

## 项目目标

构建一套可演示、可扩展、适合本科毕业设计说明书撰写的完整系统：

1. Web 前端负责用户交互。
2. Linux C++ 服务负责高并发接入。
3. Python FastAPI 服务负责认证、会话、日志和模型推理。
4. MySQL 负责持久化设计。
5. 文档负责论文、运行、测试与图表落地。

## 第一版边界

- 优先打通主链路：登录、提问、返回答案、保存历史。
- 管理功能保留最小可用闭环：查看日志、查看和更新配置。
- C++ 网关给出可编译的 Linux 骨架与核心实现，便于后续继续深挖性能。
